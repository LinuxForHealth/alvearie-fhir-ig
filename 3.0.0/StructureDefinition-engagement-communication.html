Resource Profile: Engagement Communication

Name:EngagementCommunication
Title:Engagement Communication
Status:Active as of 2021-10-07T14:44:30+00:00
Definition:

This profile is a record of an engagement communication to a patient including successful, planned, and failed communications.

Source Resource:XML / JSON / Turtle

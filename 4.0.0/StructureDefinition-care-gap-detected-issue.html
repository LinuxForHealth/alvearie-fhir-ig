Resource Profile: Care Gap Detected Issue

Defining URL:http://ibm.com/fhir/cdm/StructureDefinition/care-gap-detected-issue
Version:4.0.0
Name:CareGapDetectedIssue
Title:Care Gap Detected Issue
Status:Active as of 2021-10-26T02:43:06+00:00
Definition:

Detected issues to record a history of open and closed care gaps identified for a patient over time.

Source Resource:XML / JSON / Turtle

The official URL for this profile is:

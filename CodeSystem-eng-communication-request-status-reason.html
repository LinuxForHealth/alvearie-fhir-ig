CodeSystem: Engagement Communication Request Status Reason Codes

Official URL: http://ibm.com/fhir/cdm/CodeSystem/eng-communication-request-status-reason Version: 7.0.0
Active as of 2022-09-06 Computable Name: EngagementCommunicationRequestStatusReasonCodeSystem

Status codes detailing the reason and status history for an Engagement communication request.

This Code system is referenced in the content logical definition of the following value sets:

  • EngagementCommunicationRequestStatusReasonValueSet

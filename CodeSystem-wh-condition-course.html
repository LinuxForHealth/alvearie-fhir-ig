CodeSystem: Health Data Connect Condition Course Codes

Official URL: http://ibm.com/fhir/cdm/CodeSystem/wh-condition-course Version: 7.0.0
Active as of 2022-09-07 Computable Name: WhConditionCourseCodeSystem

Health Data Connect codes for clinical course of a disease (how the disease behaves over time). These are part of SNOMED course parent concept (288524001).

This Code system is referenced in the content logical definition of the following value sets:

  • WhConditionCourseValueSet
